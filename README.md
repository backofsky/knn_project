# KNN Classifier: бинарная и многоклассовая классификация с использованием метода k-ближайших соседей

## Пояснения
В рамках этого проекта мы реализуем алгоритм машинного обучения - классификатор на основе метода K-ближайших соседей. 
Реализованный алгоритм будет использоваться для задач бинарной и многоклассовой классификации.
Для выбора оптимального количества соседей будет использоваться кросс-валидация.
